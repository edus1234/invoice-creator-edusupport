# 請求書自動作成システム（個人事業主対応版）

## プロジェクト概要
- **プロジェクト名**: 請求書自動作成システム
- **対象**: 個人事業主・フリーランス・小規模事業者
- **形式**: ワンページ形式（上部フォーム・下部プレビュー）
- **目標**: 業務委託の報酬請求を効率化する直感的なツール

## 🎯 主な機能（完全実装済み）

### ✅ 基本情報入力
- **請求元・請求先**の会社名（お名前）入力
- **インボイス登録番号**入力（任意・インボイス制度対応）
- **請求書番号**（任意・自動生成も可能）
- **発行日**設定

### ✅ 作業項目の管理
- **日付、作業内容、時間、時給**を入力
- 各項目の**金額が自動計算**（時給 × 時間）
- 「**項目を追加**」ボタンで複数の項目を登録可能
- 不要な項目は**削除可能**

### ✅ 自動計算（税込み・税抜き対応）
- **税抜き計算**: 従来通り（小計に税率をかける）
- **税込み計算**: 時給が税込み価格として入力される
- 各項目の金額が**即座に計算**されます
- 全項目の**合計請求額も自動で表示**

### ✅ PDF出力
- 「**PDFとしてダウンロード**」ボタンをクリック
- **ブラウザの印刷画面**が開きます
- 「送信先」で「**PDFに保存**」を選択してダウンロード

### ✅ データ保存機能
- **一時保存**: 入力内容をブラウザに自動保存
- **保存データ読込**: 以前の入力内容を復元
- **データクリア**: 保存データと入力内容をリセット

## 📱 使い方

1. **基本情報の入力**
   - 請求元：自分の名前または会社名
   - インボイス登録番号：あれば入力（なくてもOK）
   - 請求先：クライアントの名前または会社名
   - 発行日：請求書の発行日

2. **計算方式の選択**
   - **税抜き計算**：時給が税抜き価格の場合
   - **税込み計算**：時給が税込み価格の場合

3. **作業項目の入力**
   - 作業した日付
   - 作業内容の説明
   - 作業時間（0.5時間刻み）
   - 時給（税込み/税抜きは設定による）
   - 金額は自動で計算されます

4. **データ保存**
   - 「一時保存」で入力内容を保存
   - 次回アクセス時に「保存データ読込」で復元可能

5. **PDF出力**
   - 「PDFとしてダウンロード」ボタンでPDF化
   - ブラウザの印刷機能を使用して高品質なPDF出力

## 💡 個人事業主向けの特徴

### 🎨 シンプルなデザイン
- **担当者欄を削除**し、名前/会社名のみの入力
- **不要なメッセージを排除**したクリーンなレイアウト
- **直感的な操作**で迷わず使える

### 📊 柔軟な計算方式
- **税抜き計算**：時給3000円 + 10% = 3300円
- **税込み計算**：時給3300円（税込み）→ 小計3000円 + 税300円

### 🏢 インボイス制度対応
- **インボイス登録番号**欄を追加
- 適格請求書の要件に対応
- 番号がない場合は空白でOK

### 💾 データ永続化
- **ブラウザ内保存**でプライバシー保護
- **自動保存機能**で入力内容を保護
- **簡単復元**で作業効率UP
