# 請求書自動作成システム（金融機関対応版）

## プロジェクト概要
- **プロジェクト名**: 請求書自動作成システム
- **対象**: 業務委託・フリーランス・個人事業主
- **形式**: 業務委託標準レイアウト（金融機関全対応）
- **目標**: あらゆる金融機関に対応したプロフェッショナル請求書作成

## 🎯 主な機能（完全実装済み）

### ✅ 多様な金融機関対応
- **8種類の金融機関**：銀行・信用金庫・信用組合・労働金庫・農協・郵便局・ネット銀行・その他
- **プルダウン選択式**で簡単選択
- **振込手数料負担明記**で支払い条件を明確化

### ✅ 標準的な請求書レイアウト
- **請求先を最初に大きく表示**（「～様」付きで丁寧な表記）
- **発行者情報を下部に配置**（業務委託の標準形式）
- **振込先口座情報を詳細表示**
- **支払期限の自動計算**（発行日翌月末まで）

### ✅ 基本情報入力
- **請求先**: 会社名（お名前）「～様」付き表示
- **発行者**: 自分の名前または会社名、インボイス登録番号対応
- **振込先口座**: 金融機関種別・機関名・支店名・口座種別・口座番号・名義
- **支払期限**: 発行日から翌月末まで自動計算

### ✅ 作業項目の管理
- **日付、作業内容、時間、時給**を入力
- 各項目の**金額が自動計算**（時給 × 時間）
- 「**項目を追加**」ボタンで複数の項目を登録可能
- 不要な項目は**削除可能**

### ✅ 自動計算（税込み・税抜き対応）
- **税抜き計算**: 従来通り（小計に税率をかける）
- **税込み計算**: 時給が税込み価格として入力される
- **支払期限**: 発行日翌月末まで自動設定
- 全項目の**合計請求額も自動で表示**

### ✅ PDF出力
- 「**PDFとしてダウンロード**」ボタンをクリック
- **ブラウザの印刷画面**が開きます
- 「送信先」で「**PDFに保存**」を選択してダウンロード

### ✅ データ保存機能
- **一時保存**: 入力内容をブラウザに自動保存
- **保存データ読込**: 以前の入力内容を復元
- **データクリア**: 保存データと入力内容をリセット

## 📱 使い方

### 1. **基本情報の入力**
- **請求先**：クライアントの名前または会社名（「～様」が自動付与）
- **発行者**：自分の名前または会社名
- **インボイス登録番号**：あれば入力（なくてもOK）
- **発行日**：請求書の発行日（支払期限が自動計算される）

### 2. **振込先口座情報（新機能）**
- **金融機関種別**：プルダウンから選択
  - 銀行
  - 信用金庫
  - 信用組合
  - 労働金庫
  - 農業協同組合
  - 郵便局（ゆうちょ銀行）
  - ネット銀行
  - その他
- **金融機関名**：○○銀行 / ○○信用金庫等
- **支店名**：△△支店 / ○○営業部
- **口座種別**：普通・当座・貯蓄
- **口座番号**：7桁の数字
- **口座名義**：カタカナ表記
- **振込手数料**：「振込手数料はご負担願います」を自動表記

### 3. **計算方式の選択**
- **税抜き計算**：時給が税抜き価格の場合
- **税込み計算**：時給が税込み価格の場合

### 4. **作業項目の入力**
- 作業した日付
- 作業内容の説明
- 作業時間（0.5時間刻み）
- 時給（税込み/税抜きは設定による）
- 金額は自動で計算されます

### 5. **データ保存と管理**
- 「一時保存」で入力内容を保存
- 次回アクセス時に「保存データ読込」で復元可能

### 6. **PDF出力**
- 「PDFとしてダウンロード」ボタンでPDF化
- 業務委託標準レイアウトで美しい仕上がり
