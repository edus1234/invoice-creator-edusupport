# 請求書自動作成システム（業務委託標準版）

## プロジェクト概要
- **プロジェクト名**: 請求書自動作成システム
- **対象**: 業務委託・フリーランス・個人事業主
- **形式**: 業務委託標準レイアウト（請求先上部・発行者下部）
- **目標**: プロフェッショナルな請求書作成の完全自動化

## 🎯 主な機能（完全実装済み）

### ✅ 標準的な請求書レイアウト
- **請求先を最初に大きく表示**（「～様」付きで丁寧な表記）
- **発行者情報を下部に配置**（業務委託の標準形式）
- **振込先口座情報を詳細表示**
- **支払期限の自動計算**（発行日翌月末まで）

### ✅ 基本情報入力
- **請求先**: 会社名（お名前）「～様」付き表示
- **発行者**: 自分の名前または会社名、インボイス登録番号対応
- **振込先口座**: 銀行名・支店名・口座種別・口座番号・名義
- **支払期限**: 発行日から翌月末まで自動計算

### ✅ 作業項目の管理
- **日付、作業内容、時間、時給**を入力
- 各項目の**金額が自動計算**（時給 × 時間）
- 「**項目を追加**」ボタンで複数の項目を登録可能
- 不要な項目は**削除可能**

### ✅ 自動計算（税込み・税抜き対応）
- **税抜き計算**: 従来通り（小計に税率をかける）
- **税込み計算**: 時給が税込み価格として入力される
- **支払期限**: 発行日翌月末まで自動設定
- 全項目の**合計請求額も自動で表示**

### ✅ PDF出力
- 「**PDFとしてダウンロード**」ボタンをクリック
- **ブラウザの印刷画面**が開きます
- 「送信先」で「**PDFに保存**」を選択してダウンロード

### ✅ データ保存機能
- **一時保存**: 入力内容をブラウザに自動保存
- **保存データ読込**: 以前の入力内容を復元
- **データクリア**: 保存データと入力内容をリセット
